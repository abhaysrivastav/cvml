## Projects

### Image Captioning Project
Located in `image-captioning-project/` directory.

An image captioning project that uses deep learning to automatically generate captions for images. The project includes:
- Dataset loading and preprocessing
- Model training using CNN-RNN architecture
- Inference capabilities
- Jupyter notebooks for step-by-step implementation

See [image-captioning-project/README.md](image-captioning-project/README.md) for detailed instructions.
